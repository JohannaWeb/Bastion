Welcome to Aurora!

This is a demonstration of the newly modernized Aurora layout engine. We have completely rewritten the text rendering algorithms to achieve perfect 1:1 aspect ratio mapping between the layout bounding boxes and the GPU painter.

Flexbox Layouts: The navigation bar above uses display: flex with justify-content: space-between and align-items: center. The navigation items utilize display: inline-block.

Notice how the text wrapping gracefully cascades without intersecting adjacent lines, and the bold text correctly allocates horizontal padding instead of infinitely bleeding into the following characters!
